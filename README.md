Project Workflow

1. Launch browser using BaseTest
2. Open required URL (Forms / Alerts / Frames / Tables)
3. Initialize Page class object
4. Locate elements using Page Object Model
5. Perform actions (click, type, select)
6. Handle alerts and frames using switch methods
7. Upload file using sendKeys
8. Capture data / validate results
9. Execute tests using TestNG (testng.xml)
10. Close browser after execution
